# 📊 Crypto Stats App (Flutter)

A modern, high-performance cryptocurrency tracking application built with Flutter.
It provides real-time market data, interactive charts, and a smooth UI experience.

---

## ✨ Features

* 🔍 Real-time cryptocurrency search
* 📈 Interactive price charts (24h, 7d, 30d, 3m, 1y)
* 💰 Market statistics (Market Cap, Volume, BTC dominance)
* ⚡ Infinite scrolling with pagination
* 📊 Detailed coin analytics screen
* 🎯 Smooth UI with shimmer loading states
* 🔄 Pull-to-refresh support

---

## 🧠 Architecture

This app follows a **clean and scalable structure**:

* GetX for state management
* MVVM-inspired separation (View / ViewModel / Model)
* API-driven architecture using Dio

---

## 🛠 Tech Stack

* Flutter (UI framework)
* Dart (language)
* GetX (state management)
* Dio (HTTP client)
* FL Chart (data visualization)
* Intl (formatting)

## 🚀 Getting Started

### 1. Clone the repository

```bash id="clone01"
git clone https://github.com/AbdulManan-official/crypto-stats-flutter.git
```

### 2. Install dependencies

```bash id="deps01"
flutter pub get
```

### 3. Run the app

```bash id="run01"
flutter run
```

---

## 📦 Build APK

```bash id="apk01"
flutter build apk --release
```
